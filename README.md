# Run and Eat

A specialized platform for discovering and sharing gastronomic events worldwide.

## ğŸ“– About The Project

Run and Eat is a web application designed to connect food enthusiasts with local gastronomic events. Whether you're a tourist looking for authentic culinary experiences or a local wanting to discover new food events, Run and Eat makes it easy to find, share, and participate in gastronomic activities.

### The Problem

Tourists and locals often struggle to find gastronomic events such as food fairs, tapas routes, food trucks, or culinary cultural events in their area. Small-scale events lack visibility, and existing platforms are too generic.

### The Solution

Run and Eat provides a dedicated platform exclusively for food-related events, where anyone can:
- **Discover** gastronomic events near them
- **Create** and share their own food events
- **Participate** and connect with other food lovers
- **Review** events and build a trusted community

## ğŸš€ Getting Started

### Prerequisites

- Web server (Apache, Nginx, etc.)
- PHP 7.4 or higher (for backend functionality)
- Modern web browser

### Installation

1. Clone the repository
```bash
git clone https://github.com/ignaciobrenas/Run-Ea
```

2. Navigate to the project directory
```bash
cd run-and-eat
```

3. Open `index.html` in your browser or set up a local server

For PHP functionality:
```bash
php -S localhost:8000
```

## Usage

### For Participants:
1. Register as a participant
2. Browse events on the interactive map
3. Filter events by your preferences
4. Sign up for events
